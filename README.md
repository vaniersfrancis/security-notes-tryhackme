# TryHackMe Notes – Key Concepts Learned

## Overview
These notes summarize key concepts learned while completing the TryHackMe Pre Security path. The focus has been on building a strong foundation in networking, systems, and core security principles.

---

## Networking Fundamentals

### Key Concepts
- IP addresses identify devices on a network  
- DNS translates domain names into IP addresses  
- Ports allow different services to run on a single device  
- Common ports:
  - 80 → HTTP  
  - 443 → HTTPS  
  - 22 → SSH  
  - 53 → DNS  

### What I Learned
Understanding how devices communicate across networks and how services are exposed through ports is critical for identifying potential vulnerabilities and troubleshooting connectivity issues.

---

## Operating Systems

### Key Concepts
- Windows and Linux are the most commonly used operating systems  
- The command line allows direct interaction with the system  
- File systems organize and store data  

### What I Learned
Gained a better understanding of how systems are structured and how users and processes interact with the operating system.

---

## Security Principles

### CIA Triad
- Confidentiality → Protect data from unauthorized access  
- Integrity → Ensure data is not altered  
- Availability → Ensure systems and data are accessible  

### What I Learned
The CIA triad is the foundation of security. Most security decisions and controls are based on maintaining one or more of these principles.

---

## Risk, Threat, and Vulnerabilities

### Key Concepts
- Vulnerability → Weakness in a system  
- Threat → Potential danger  
- Risk → Likelihood of a threat exploiting a vulnerability  
- Exploit → Method used to take advantage of a vulnerability  

### What I Learned
Understanding the relationship between these concepts is essential for identifying and reducing security risks.

---

## Authentication & Access Control

### Key Concepts
- Authentication verifies identity  
- Authorization determines access level  
- Multi-Factor Authentication (MFA):
  - Something you know (password)  
  - Something you have (device)  
  - Something you are (biometrics)  

### What I Learned
Strong authentication is critical in preventing unauthorized access and protecting systems.

---

## General Takeaways

- Security is built on understanding how systems and networks operate  
- Many vulnerabilities exist due to misconfigurations or weak controls  
- Hands-on labs helped reinforce how theoretical concepts apply in real environments  
- Troubleshooting and security are closely related—understanding system behavior helps identify issues faster  

---

## Defensive Security Concepts

### Key Points
- Monitoring systems helps detect suspicious activity  
- Logs provide visibility into system behavior  
- Security teams analyze alerts to identify threats  

### What I Learned
Understanding how to monitor and analyze system activity is a key part of defensive security and SOC roles.
